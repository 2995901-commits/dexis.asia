Кто за этим стоит

Виталий Ли

DEXIS основан в Алматы. До создания компании Виталий вёл маркетинг в Vitis Capital — инвестиционной компании с офисами в Алматы, Астане и Сеуле, работающей с регуляторной лицензией AIFC. Этот опыт дал понимание того, как принимаются решения в среде, где данные служат материалом для управления, а не приложением к нему.

DEXIS строится на наблюдении, которое сложно увидеть со стороны: казахстанский средний бизнес уже зрелый по управленческой культуре, но инструменты для работы с данными у него остались на уровне 2010-х. Excel-таблицы, разрозненные конфигурации 1С, отчётность с задержкой в неделю. Этот разрыв и есть тот вакуум, который DEXIS заполняет.

Подход

DEXIS внедряется лично основателем — без отдела продаж, без посредников, без передачи проекта команде после подписания. Это решение продуктивное и осознанное: каждое внедрение даёт обратную связь, которая возвращается в продукт. Компании, которые работают с DEXIS сегодня, фактически формируют его архитектуру.

Когда DEXIS вырастет до команды, этот подход изменится. Сейчас он — главное преимущество для первых клиентов.
